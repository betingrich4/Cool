WhatsApp Status Bot

Automatically view and react to WhatsApp status updates

How to Get Your Session ID

  1. Run the command .getid in your existing WhatsApp bot
  2. You'll receive a session ID in format: Demo-Slayer~FILEID#DECRYPTKEY
  3. Copy and paste it exactly as shown above

Note: Your bot will automatically react to all status updates with random emojis.

Your session will remain active for 24 hours after deployment
